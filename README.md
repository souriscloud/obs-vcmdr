# obs-vcmdr | VueJS App for controling OBS using OBS-WebSocket plugin.

using `obs-websocket-js` library

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
